9th International Conference on Data Analytics and Cyber Security (DACS 2026)

December 21-23, 2026

Rourkela, India

National Institute of Technology (NIT) Rourkela

Call for Papers

DACS 2026 invites researchers and practitioners to submit original and high-quality papers. Accepted papers will be published in the conference proceedings and indexed in major databases.

Track 1: Data Analytics

  • Machine learning and AI for data analysis
  • Big data analytics and distributed computing
  • Data mining and knowledge discovery
  • Data visualization and decision support systems
  • Applications of data analytics in various domains

Track 2: Cyber Security

  • Network security and intrusion detection
  • Cryptography and access control
  • Malware analysis and threat intelligence
  • Risk assessment and vulnerability management
  • Cyber security in cloud computing and IoT
  • Cyber-threat intelligence and incident response
  • Blockchain and distributed ledger technologies
  • Zero-trust security models
  • Quantum cryptography and post-quantum security

Track 3: Synergies in Analytics & Security

  • Data-driven threat detection and analysis
  • Data-driven security incident response
  • Data-driven risk management
  • Security intelligence and threat hunting
  • Behavioral anomaly detection
  • Security automation and orchestration
  • Privacy considerations in data analytics

Track 4: Emerging Technologies

  • Large Language Models (LLM)
  • Generative AI in Security
  • Cloud Computing and Distributed Systems
  • Internet of Things (IoT) and Edge Computing
  • Human-Computer Interaction (HCI)
  • Quantum Computing

Awards & Publications

Three best paper awards will be presented: Best Regular Paper, Runner-up, and Best Student Paper.
