Call for Papers

DACS invites researchers and practitioners to submit original and high-quality papers addressing themes including, but not limited to:

Track 1: Data Analytics

  • Machine learning and AI for data analysis
  • Big data analytics and distributed computing
  • Data mining and knowledge discovery
  • Data visualization and decision support

Track 2: Cyber Security

  • Network security and intrusion detection
  • Cryptography and access control
  • Malware analysis and threat intelligence
  • Blockchain and zero-trust models

Track 3: Synergies & Synergies

  • Data-Driven Threat Detection
  • Behavioral Anomaly Detection
  • Security Intelligence and Threat Hunting
  • Privacy in data analytics

Track 4: Emerging Technologies

  • Large Language Models (LLM)
  • Generative AI
  • Internet of Things (IoT) and Edge Computing
  • Quantum Computing
